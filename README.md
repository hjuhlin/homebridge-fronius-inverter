# Homebridge Platform Fronius Inverters Plugin

This is a plugin for Fronius Inverters

# Support for

1.0.2

- First version

# Default config

```json
"platforms": [
    {
        "name": "Fronius Inverter Energy",
        "ip": "192.168.0.X",
        "UpdateTime": 5,
        "MaxProduction": 10000,
        "ViewElectricPowerProduction": false,
        "EveLoging": false,
        "Debug": false,
        "platform": "FroniusInverterEnergy"
    }
]
```
